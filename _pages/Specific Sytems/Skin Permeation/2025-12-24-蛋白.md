# 建模细节

## KLK5/KLK7的糖基化与建模取舍

### N-糖位点总览（UniProt 2025-06-18）

| 蛋白 | UniProt | 成熟链范围 | 注释的N-糖位点 |
|------|---------|-------------|----------------|
| KLK5 | Q9Y337 | 23-293 | **Asn69、Asn173、Asn208、Asn252** |
| KLK7 | P49862 | 23-252 | **Asn246** |

- KLK5的四个`N-X-S/T`位点均位于催化域外围loop，其中**N208在2PSX/6QFE晶体结构中实测到GlcNAc残基**，说明折叠时高度占位。N69/N173接近N端低复杂度区，N252位于C端β折叠。
- KLK7只有Asn246具备保守基序，皮肤glycoproteomics多次检测到占位。

### 在细胞间脂质环境中是否都要保留？

| 场景 | 建议 | 理由 |
|------|------|------|
| **原子级精细MD** | KLK5至少保留**N208与N252**的Man₃GlcNAc₂；N69/N173视实验设定可选。KLK7默认保留N246。 | 保证最关键的loop-糖网络存在，避免夸大正电吸附并保持实际的柔性。 |
| **粗粒化或大规模筛查** | 可先**全部去糖**，然后在关键体系上逐步补回。 | 降低自由度，快速评估上限。完备体系用于验证结果稳健性。 |
| **去糖/突变对照** | 同时搭建“保留关键糖链”与“去糖”两套模型。 | 便于量化糖链对HA吸附、脂质嵌入的贡献。 |

### 忽略糖链的风险

1. **静电放大**：KLK5/7表面正电主要集中在loop区域，去糖会显著增强与HA/脂质头基的静电吸附，得到偏强结果。
2. **结构松散**：N208/N252邻近催化沟槽，糖链缺失后loop更易摆动，可能出现非生理构象。
3. **实验吻合度**：角质层蛋白质谱显示成熟KLK5/7仍携带N-糖，完全去糖仅适合作为对照。

**结论**：在细胞间脂质环境的分子建模中，不必一次性添加全部糖链，但**至少保留KLK5的N208/N252和KLK7的N246**可避免闯入非真实的强吸附状态。剩余位点可按研究目标与算力灵活启用，并通过去糖对照辨析糖链对HA/脂质相互作用的具体贡献。

### 必要的序列截取与晶体结构模板

| 项目 | KLK5 | KLK7 |
|------|------|------|
| UniProt ID | Q9Y337 | P49862 |
| 前导信号肽 | 1-22 | 1-22 |
| **活性前体链** | 23-293（含37 aa激活肽） | 23-252（含7 aa激活肽） |
| **成熟催化链** | **67-293**（Ile67起始） | **30-252**（Ile30起始） |
| 可用晶体结构 | 2PSX/2PSY、6QFE、7U5B（均为67-293） | 1O6I、2QXI（Ile30起始） |
| 主要PTM（除糖链外） | 6对二硫键：73-206、93-109、178-279、185-251、217-231、241-266 | 6对二硫键：75-193、94-109、145-220、155-201、179-243、215-241 |

- **序列选择**：在细胞间脂质模拟中，应去除信号肽与激活肽，直接使用成熟催化域（KLK5: 67-293；KLK7: 30-252）。若要表现层板小体内的前体状态，可保留激活肽但需额外约束（因其与催化域仅松散接触）。
- **晶体结构**：KLK5的多套高分辨率结构覆盖了67-293的催化域，可直接作为初始坐标；KLK7也有2 Å级别结构（1O6I、2QXI）可用。若需包含glycan，可使用AlphaFold或CHARMM-GUI对上述PDB进行糖基化。

### 其他必需的PTM/约束

1. **二硫键**：KLK5/7均含6对高度保守的二硫键，缺一不可。建模时需在力场中显式定义S-S约束，否则长时间模拟会出现loop塌陷。
2. **N端新生胺**：成熟链经切割后在Ile67（KLK5）或Ile30（KLK7）形成新的N端胺，带正电。请确认拓扑文件采用带正电的`ACE/NH3+`处理。
3. **蛋白酶活化态**：KLK5/7在酸性条件下由KLK14或自身切割激活，pH 5-6的细胞间脂质环境应默认采用活化态（即无激活肽、催化三联体组装完毕）。
4. **金属抑制（可选）**：KLK5在His150附近有Zn²⁺结合位点，可在含Zn抑制的场景加入一个Zn²⁺并按照PDB 2PSX的配位几何加以约束。

> **建模流程小结**：去除信号肽与激活肽 → 加入必要的二硫键与N-糖链 → 根据模拟目标选择是否加载Zn²⁺/抑制剂 → 将成熟催化域放置于脂质-水界面，执行能量最小化和短程平衡，再进入长程生产阶段。

## β-Glucocerebrosidase（GBA）的建模要点

| 项目 | 数据/说明 |
|------|-----------|
| UniProt | P04062 |
| 信号肽 | 1-39（经ER/高尔基切除） |
| **成熟链** | **40-536**（常以40-497或40-517的晶体结构表示） |
| 主要晶体结构 | 1OGS、2NT0、3GXI、5O1U（均覆盖成熟结构，部分带抑制剂或糖链） |
| N-糖位点 | Asn58、Asn98、Asn185、Asn309、Asn501（MS证实；前三个位点占位度最高） |
| 二硫键 | Cys43-Cys55、Cys57-Cys62（稳定N端β折叠） |
| 催化残基 | Glu235（酸催化）、Glu340（亲核）、Asp127（质子供体/调节） |

### 序列与模板选择
- 角质层来源的GBA已经去除信号肽，并以成熟单体非共价成对形式存在。模拟时直接使用残基40-536即可；如需包含前体或突变，可借助AlphaFold（AF-P04062-F1）或SWISS-MODEL补足缺失loop。
- 推荐使用PDB 1OGS（1.75 Å，含dUDP衍生物）、2NT0（与抑制剂NNC存在）、3GXI/5O1U（包含不同配体）作为初始坐标。若需无配体构象，可删除晶体配体后重建局部氢键。

### PTM与约束配置
1. **N-糖链**：在角质层环境中GBA仍保留至少三条N-糖。建模时建议：
   - 默认加载Asn58、Asn98、Asn185上的核心Man₃GlcNAc₂；
   - Asn309位于表面疏水面，可根据体系大小选择性添加；
   - Asn501更靠近C端尾部，如模拟胶束/溶液可保留，若体系拥挤可视作截断。
2. **二硫键**：Cys43-Cys55、Cys57-Cys62必须在力场中固定，否则N端β折叠会塌陷影响活性位点入口。
3. **pH与质子化**：GBA在pH 5.2-5.6最活跃。建模酸性SC下层时，将Glu235/Glu340设为去质子化，Asp127保持单负电，即可维持催化三联体；若模拟更酸 (pH<4.5) 则需考虑单次质子化以防数值不稳。
4. **二聚化与配体**：晶体常显示非共价二聚体（界面位于N端loop）。若体系只包含一个单体，应在界面处施加轻度约束或在模拟盒中加入第二个单体，避免N端无支撑。角质层环境中通常与神经酰胺前体/游离脂肪酸接触，可按需求放入短链GlcCer或脂质头基。

### 在细胞间脂质环境中的具体建议
1. **放置方式**：将GBA对接在脂质双层或13 nm周期堆叠的水层-脂质交界面，使N-糖面向水相；C端疏水斑块（残基300–340附近）略微贴近脂质头基，模拟层板小体释放后的定位。
2. **质子化/离子条件**：在pH 5.2条件下设置质子化后，使用0.15 M NaCl或NaOAc平衡电荷；若考虑Ca²⁺调节，可在C端引入一个结构Ca²⁺（某些结构中存在）。
3. **糖链策略**：先在最小系统中评估“无糖 vs. N58/N98/N185 + Man₃GlcNAc₂”的差异，再依据需要扩展至复杂型糖链或额外糖位点，用于研究糖链对HA或脂质结合的屏蔽作用。
4. **生产模拟流程**：最小化 → NVT预热至310 K → NPT平衡（≥20 ns）→ 生产阶段（≥200 ns）以监测GBA表面电场、HA短暂停留等指标。

> **建模重点**：GBA本质是高丰度的酸性水解酶。即便在HA研究中扮演“对照”角色，也应保留其核心PTM与酸性构象，以免因去糖/去二硫而过度放大与HA的静电作用或低估局部酸性环境的影响。
