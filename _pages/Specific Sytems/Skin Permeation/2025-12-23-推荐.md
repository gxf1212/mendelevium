# 角质层蛋白筛选报告

## 最终建议

**筛选原则**：无直接脂质结合证据也可接受，文献未报道HA结合功能没关系，只要带正电，重点是**真的能游离到细胞间脂质**（角质层中外层）且**丰度够高**。

### 候选蛋白优先级与关键优缺点

**说明**：按优先级降序排列。⭐越多优先级越高，⭐表示不推荐（作为对照）。

| 候选蛋白 | 优先级 | 核心优势 | 关键局限 |
|---------|-------|---------|---------|
| **KLK5/KLK7** | ⭐⭐⭐⭐⭐ | • **丰度绝对最高**（11-67 mg/g，比ASM高100-220倍）<br>• 明确的脂质-水界面定位<br>• pI 7.8-8.3，pH 5-6时保持中等正电<br>• 高浓度保证充分接触，累积效应显著 | • 缺少直接脂质结合模体<br>• "暂时附着"是否长时间嵌入脂质层存疑<br>• 蛋白酶功能可能降解HA-蛋白复合物 |
| **β-defensins** | ⭐⭐⭐⭐⭐ | • **结构最优**：强阳离子(+11)+两亲性+脂质结合<br>• 明确的细胞间脂质定位<br>• 极高pI（>9），pH 4-6全程强正电<br>• 炎症时上调2-5倍 | • **丰度低**（0.077-0.195 µg/g，比KLK7低5万倍）<br>• 是否长时间停留在脂质层缺少定量数据<br>• 本职是抗菌，非HA调控 |
| **Cathepsin L2（SCTP）** | ⭐⭐⭐⭐ | • 角质层细胞间隙的主导胱天蛋白酶<br>• pI≈8.5，pH 5-6保持较强正电<br>• TSH 100 µU/mL即可在30 min内诱导分泌 | • 绝对蛋白量仍缺精确定量<br>• 主要功能是降解角质胞桥，可能破坏复合物<br>• 活性受Cystatin M/E严格调控 |
| **GBA** | ⭐⭐⭐⭐ (高丰度对照) | • **高丰度**（1,530 µg/g，与KLK5下限相当）<br>• 脂质代谢主力酶，功能必需<br>• 丰度是ASM的5-6倍<br>• pI 7.29，pH 4.5-5.5时带正电 | • **与HA无直接关系**（脂质代谢，无HA结合报道）<br>• 正电荷密度远低于pI>8的蛋白<br>• 活性位点被脂质底物占据 |
| **sPLA₂-IIF** | ⭐⭐⭐ (待确认) | • **最高pI**（9.3-9.4），pH 4-6全程强正电<br>• 唯一的环境调控机制（pH+脂质）<br>• knockout严重屏障缺陷 | • **蛋白定量数据完全缺失**（仅RNA nTPM 10-20）<br>• **底物问题**：角质层磷脂<0.2%，仅在SG-SC交界<br>• 推测丰度10-500 µg/g（需验证） |
| **ASM** | ⭐⭐⭐ | • pI≈6.7，pH 4.5-5.2时保持弱正电<br>• 丰度240-310 µg/g（已明确定量）<br>• 分泌到角质层下三分之一的细胞间脂质 | • **pH敏感**：pH>5.5时净电荷趋近0<br>• 丰度比KLK低100-220倍<br>• 主要功能是鞘磷脂→神经酰胺，缺乏直接HA结合证据<br>• 定位偏向SG/SC交界 |
| **LL-37** | ⭐⭐⭐ (炎症) | • 最强脂质结合（两性螺旋）<br>• 净电荷+6，酸性pH下更高<br>• 膜扰动可能促进渗透<br>• 炎症时达20 µg/mL | • **稳态皮肤几乎检测不到**<br>• 是否长时间停留脂质层无定量数据<br>• 仅限病理状态 |
| **CDSN 15 kDa片段** | ⭐⭐ | • KLK5/7降解后仍保留pI>8的强正电核心<br>• 末端柔性肽段可探入脂质-水界面<br>• 与角化包膜/角质胞桥交界的"蛋白舌"可短暂触及HA | • **空间受限**：固定在角质胞桥边缘，无法远距离扩散<br>• 需要KLK级联后才暴露，时空窗口短<br>• 无游离态，仅微域作用 |

## 候选蛋白质清单

基于定位相关性、结构适配性和机制匹配三个维度的综合评估，以下**8种蛋白质**最有可能在角质层细胞间隙与HA/OP发生分子层面相互作用：

1. **β-defensins (hBD-2/hBD-3)** - 阳离子抗菌肽（结构最优）
2. **Kallikrein 5/7 (KLK5/KLK7)** - 最高丰度可溶性蛋白酶
3. **Cathepsin L2（SCTP）** - 角质层主导胱天蛋白酶
4. **GBA (β-葡糖神经酰胺酶)** - 高丰度脂质代谢酶（pH窗口受限，作为对照）
5. **sPLA₂-IIF (PLA2G2F)** - 碱性脂质代谢酶（最高pI，丰度未知）
6. **酸性鞘磷脂酶 (ASM)** - 角质层酸化的上游脂质酶
7. **LL-37 (Cathelicidin)** - 两性螺旋阳离子肽（仅限炎症）
8. **Corneodesmosin 15 kDa片段** - 角质胞桥边缘的正电"蛋白舌"（空间受限）

### 关键事实

**核心发现**：文献调研揭示，几乎所有HA特异性受体和结合蛋白（CD44、TSG-6、Versican、RHAMM、LYVE-1）**完全不存在于角质层**，它们仅分布于表皮活细胞层（基底层、棘层、颗粒层）。这一核心发现得到了Wang et al. (1992) 的直接证实："In epidermis both HA and CD44 were found in the basal and spinous cell layers, but neither was present in the stratum granulosum and stratum corneum" ([DOI: 10.1007/BF00717001](https://doi.org/10.1007/BF00717001))。

**因此**：本报告筛选的候选蛋白虽然存在于角质层细胞间隙，但它们的主要生理功能并非HA结合，而是**脱屑调控、抗菌防御和结构维持**。

## 核心证据总结

### 1. β-defensins (hBD-2/hBD-3)

| 维度 | 核心证据 |
|-----|---------|
| **定位** | ✅ 细胞间脂质（储存于层板小体，随脂质层排出到脂质-水界面）<br>• 健康人角质层：77-195 pg/µg protein<br>• AD患者：180-890 pg/µg protein（炎症时2-5倍上调） |
| **理化性质** | ✅ 强阳离子：hBD-3净电荷+11（富含Lys/Arg）<br>✅ 两亲性结构：疏水斑块+正电区，可嵌入脂质双层<br>✅ pI值高：酸性pH下完全质子化 |
| **脂质结合** | ✅ 中等（可与阴离子脂质PS/PI结合，定位于脂质-水界面） |
| **功能** | ⚠️ 主要作用是**抗菌和免疫调节**，非HA特异性结合<br>✅ 可能的"桥接"作用：HA-β-defensin-脂质三元复合物 |
| **关键文献** | Clausen et al., 2018 (Tape stripping定量)；Oren et al., 2003 (层板小体储存) |

### 2. sPLA₂-IIF (PLA2G2F)

| 维度 | 核心证据 |
|-----|---------|
| **定位** | ✅ 细胞间脂质（上颗粒层和角质层下部，与层板小体共转运）<br>• RNA表达：Skin组织nTPM 10-20<br>• 组织特异性：Cluster 58 "Skin - Keratinization" (confidence = 1.0) |
| **理化性质** | ✅ **最高pI**：9.3-9.4（所有候选中最高）<br>✅ pH 4.5-6.0时仍保持强正电 |
| **脂质结合** | ✅ 强（直接在脂质层中催化磷脂水解，生成游离脂肪酸） |
| **功能** | ✅ **唯一的环境调控机制**：通过pH和脂质代谢影响HA行为<br>✅ 功能必需性：PLA2G2F⁻/⁻小鼠SC pH升高0.4单位，TEWL恢复延长>50%<br>⚠️ **底物限制**：仅作用于SG-SC交界处的磷脂前体，成熟SC无磷脂 |
| **关键文献** | Human Protein Atlas验证（2025-12-23）；Mao-Qiang et al., 2010 (knockout表型) |

### 3. Kallikrein 5/7 (KLK5/KLK7)

| 维度 | 核心证据 |
|-----|---------|
| **定位** | ✅ 细胞间脂质（暂时附着于脂质/包膜表面定位底物）<br>• **KLK7**：11.1-67.7 mg/g总蛋白（**角质层中丰度最高的可溶性蛋白酶**）<br>• **KLK5**：1.3-15.9 mg/g总蛋白 |
| **理化性质** | ✅ 正电：KLK5 pI≈7.8，KLK7 pI≈8.3<br>✅ pH 5-6时整体带净正电（His残基部分质子化） |
| **脂质结合** | ⚠️ 弱（无特异性脂质结合域，暂时附着） |
| **功能** | ⚠️ 主要功能是**蛋白水解**（降解CDSN/DSG1/DSC1启动脱屑）<br>⚠️ 可能降解HA-蛋白复合物<br>✅ 高浓度下可能形成静电屏蔽层 |
| **关键文献** | Kovarik et al., 2016 (质谱定量)；Descargues et al., 2006 (定位) |

### 4. Cathepsin L2（SCTP）

| 维度 | 核心证据 |
|-----|---------|
| **定位** | ✅ 细胞间脂质（角质胞桥周围，待脱屑微域）<br>• 28-30 kDa成熟酶占角质层胱天蛋白酶活性主峰<br>• TSH 100 µU/mL可在30 min内诱导分泌 |
| **理化性质** | ✅ pI≈8.5，pH 5-6保持净正电<br>✅ 疏水-阳离子混合界面，可在脂质-水界面暂时附着 |
| **脂质结合** | ⚠️ 弱（与角质胞桥周围蛋白-脂质界面暂时接触） |
| **功能** | ⚠️ 主要功能是**降解CDSN/DSC1**<br>✅ "剪刀+吸附"双重作用：先削弱角质胞桥，再凭正电捕获HA<br>✅ 剂量-时间可控（TSH诱导） |
| **关键文献** | J. Invest. Dermatol. 2003 (活性水平)；Int. J. Mol. Sci. 2020 (TSH诱导) |

### 5. 酸性鞘磷脂酶 (ASM)

| 维度 | 核心证据 |
|-----|---------|
| **定位** | ✅ 细胞间脂质（颗粒层顶端至角质层下三分之一）<br>• 丰度：39-50 µg/cm²（Tape stripping-免疫印迹定量） |
| **理化性质** | ✅ pI≈6.7，pH 4.5-5.5时呈弱正电 |
| **脂质结合** | ✅ 强（依靠疏水表面贴附在阴离子脂质头基上保持活性） |
| **功能** | ⚠️ 主要功能是**鞘磷脂→神经酰胺**（间接调控）<br>✅ 通过生成神经酰胺维持酸性pH和紧凑脂质堆叠<br>✅ 与sPLA₂-IIF互补（双酶联动调控pH和脂质组成） |
| **关键文献** | Exp. Dermatol. 2010 (定位与定量)；J. Invest. Dermatol. 1998 (绝对含量) |

### 6. LL-37 (Cathelicidin)

| 维度 | 核心证据 |
|-----|---------|
| **定位** | ✅ 细胞间脂质（两性螺旋，嵌入脂质双层）<br>• 健康皮肤：**低于检测下限**<br>• 炎症状态：20 µg/mL（银屑病） |
| **理化性质** | ✅ 阳离子：净电荷+6（pH 7），酸性pH下更高<br>✅ 两性螺旋：疏水N端+阳离子亲水端 |
| **脂质结合** | ✅ **最强**（嵌入脂质双层，形成孔道或改变膜流动性） |
| **功能** | ⚠️ 主要功能是**抗菌和伤口愈合**<br>✅ 可能的**膜扰动作用**：促进HA渗透而非阻挡<br>❌ 健康皮肤中丰度极低，实际意义有限 |
| **关键文献** | Clausen et al., 2018 (Tape stripping未检出)；Braff et al., 2005 (定位) |

### 7. CDSN 15 kDa片段

| 维度 | 核心证据 |
|-----|---------|
| **定位** | ▓ 细胞间脂质（角质胞桥-角化包膜交界，"蛋白岛"边缘）<br>• 仅在角质胞桥附近微域，无法远距离扩散 |
| **理化性质** | ✅ pI>8，pH 4-6仍保持强正电<br>✅ 富含Gly/Ser/Lys，柔性"蛋白舌"可伸入脂质-水界面 |
| **脂质结合** | ⚠️ 中等（系留于角化包膜但柔性肽段伸入脂质层） |
| **功能** | ⚠️ 主要功能是**脱屑标记**，功能丧失<br>✅ 可能的**局部静电陷阱**（角质胞桥附近微域）<br>❌ 空间受限，无法大范围影响HA渗透 |
| **关键文献** | J. Biol. Chem. 2001 (表位分析)；Br. J. Dermatol. 2004 (蛋白酶级联) |

### GBA的特殊情况：高丰度但pH窗口受限

**β-葡糖神经酰胺酶 (GBA)** 是一个重要的高丰度对照案例，值得单独说明：

| 属性 | 数据 |
|-----|------|
| **绝对丰度** | 245 µg/cm² = **1,530 µg/g总蛋白** |
| **相对于ASM** | **5-6倍**（与KLK5下限相当） |
| **相对于KLK7** | 1/7 - 1/44 |
| **pI值** | 7.29（pH 4.5-5.5时带弱正电，pH>5.5时净电荷趋近0） |
| **活性pH范围** | pH 5.2-5.6（最优活性） |
| **主要功能** | β-葡糖神经酰胺→神经酰胺的脂质代谢 |
| **定位** | 颗粒层层板小体→角质层细胞间脂质双层 |
| **活性** | 9.1 ± 0.4 nmol·min⁻¹·mg DNA⁻¹（约75%表皮β-糖苷酶集中于SC） |

**为什么GBA可作为⭐⭐⭐⭐级别的高丰度对照**：
1. **丰度优势明显**：1,530 µg/g，与KLK5下限相当，远超ASM和β-defensins
2. **定位明确**：确实分布于细胞间脂质双层
3. **在SC偏酸区带正电**：pH 4.5-5.5时（SC下层）带弱正电，理论上可与HA发生静电相互作用

**关键局限**：
1. **pH依赖性强**：当pH>5.5时（SC中上层），净电荷快速趋近0，静电相互作用减弱
2. **正电荷密度低**：pI 7.29远低于β-defensins（pI>9）和sPLA₂-IIF（pI 9.3），表面正电荷密度显著较低
3. **功能不相关**：主要催化脂质底物（β-葡糖神经酰胺），活性位点被底物占据，缺乏HA结合的文献报道
4. **作为对照的价值**：可用于验证"高丰度+弱正电"与"低丰度+强正电"对HA相互作用的相对贡献

## 蛋白丰度对比分析

### 绝对丰度定量数据

| **KLK5** | 1.3-15.9 mg/g | **1,300-15,900** | 1/5 - 1/10 | Tape strip + LC-MS/MS | ✅ 已验证 |
| **GBA** | 245 µg/cm² | **1,530** | 1/44 | Tape strip + 免疫定量 | ✅ 已验证 |

### 关键发现

#### 1. KLK7/5是绝对的丰度冠军

- **KLK7**：11.1-67.7 mg/g，比ASM高**100-220倍**
- **KLK5**：1.3-15.9 mg/g，比ASM高**5-50倍**
- 这反映了蛋白酶在脱屑调控中的核心地位
- **高浓度保证了充分的接触机会**，即使脂质结合较弱

#### 2. 脂质代谢酶丰度分层明显

- **GBA**（1,530 µg/g）> **ASM**（240-310 µg/g）> **sPLA₂-IIF**（未知）
- **GBA的丰度是ASM的5-6倍**，与其"脂质成熟主力酶"的地位相符
- ASM虽然丰度低于GBA，但仍达到百微克级，足以维持神经酰胺生成
- **脂质代谢酶通过"催化放大"效应发挥作用，不需要高丰度**

#### 3. β-defensins的"低丰度-高活性"特征

- 77-195 pg/µg = 0.077-0.195 µg/g，比KLK7低**50,000-800,000倍**
- 但抗菌活性极强（纳摩尔级IC₅₀），符合抗菌肽的典型特征
- **炎症时上调2-5倍**（180-890 pg/µg），但仍远低于KLK

#### 4. sPLA₂-IIF的数据缺口最大

- **仅有RNA表达数据**（nTPM 10-20），属于中低表达
- **蛋白绝对丰度完全未知**，这是重大研究缺口
- 推测可能在**10-500 µg/g**范围（介于Cathepsin D和ASM之间）
- 即使丰度低，但**功能必需性极高**（knockout表型严重）
- **底物限制**：角质层磷脂<0.2% w/w，可能通过"催化放大"作用，不需要高丰度
