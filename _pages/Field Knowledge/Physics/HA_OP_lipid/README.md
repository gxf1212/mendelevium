# 带电高分子与脂质膜相互作用的静电学模型

本项目系统推导了从简单点电荷模型到高分子相互作用的完整静电学理论框架。

## 文件结构

- `point_charge_model.md` - 第一章：点电荷模型基础推导
- `polymer_model.md` - 第二章：高分子链相互作用模型
- `real_system_analysis.md` - 第三章：真实体系参数与数值计算
- `figures/` - 示意图和数据分析图表
- `code/` - Python计算脚本

## 模型概述

### 第一章：点电荷模型
- 无限大带负电平板（面电荷密度 -σ）
- 正电荷 +q 固定在距离 d 处
- 负电荷 -aq 可移动，求势能最小位置

### 第二章：高分子模型
- 正电高分子：刚性直线，N个单元，固定
- 负电高分子：N个单元，可移动
- 平行结合假设与势能修正系数

### 第三章：真实体系
- 脂质双分子层体系
- 15单元正电聚合物 vs 36单元负电聚合物
- 分子模拟参数化
