# 加粗格式规范

## 基本规则

加粗只包含**正常汉字、数字、字母、基本符号**，不包含标点、括号、引号、百分号。

---

## 正确示例

### 1. 标点在加粗外

```markdown
✅ 这是**重要概念**.
✅ 发现了**显著的效果**.
✅ 使用了**贝叶斯优化**方法。
```

### 2. 百分号在加粗外

```markdown
✅ 准确率达到**92**.5%
✅ 提高了**24**%
✅ 误差在**3**%以内
```

### 3. 括号在加粗外

```markdown
✅ 使用了**贝叶斯优化**（Bayesian Optimization）方法
✅ **丝氨酸水解酶**（Serine Hydrolase）
✅ 发现了**催化三联体**（Ser-His-Asp）
```

### 4. 引号在加粗外

```markdown
✅ "**黑箱**"模型的局限性
✅ 所谓的"**过拟合**"问题
✅ 研究了"**生成式AI**"的应用
```

---

## 错误示例

### 1. 标点在加粗内 ❌

```markdown
❌ 这是**重要概念.**
❌ 发现了**显著的效果！**
❌ 使用了**贝叶斯优化**方法？
```

**为什么错误**：Markdown渲染时可能出问题，而且不符合排版规范。

### 2. 百分号在加粗内 ❌

```markdown
❌ 准确率达到**92.5%**
❌ 提高了**24%**
❌ 误差在**3%**以内
```

**为什么错误**：会导致渲染失败或格式错误。

### 3. 括号在加粗内 ❌

```markdown
❌ 使用了**贝叶斯优化（Bayesian Optimization）**方法
❌ **丝氨酸水解酶（Serine Hydrolase）**
❌ 发现了**催化三联体（Ser-His-Asp）**
```

**为什么错误**：括号内的英文不需要加粗，加粗应该只突出中文概念。

### 4. 引号在加粗内 ❌

```markdown
❌ **"黑箱"**模型的局限性
❌ 所谓的**"过拟合"**问题
❌ 研究了**"生成式AI"**的应用
```

**为什么错误**：引号是标点符号，应该在加粗外。

## 特殊情况

### 数字和单位

```markdown
✅ 温度为**300** K
✅ 能量为**-3.69** kcal/mol
✅ 时间为**15** ps
```

**规则**：数字加粗，单位不加粗。

### 中英文混合

```markdown
✅ 使用了**QM/MM方法**
✅ 采用了**DFT**计算
✅ 基于**GROMACS**软件
```

**规则**：缩写词可以加粗，但斜杠等符号也在加粗内。

### 连续加粗

```markdown
✅ **结论1**：xxx。**结论2**：xxx
✅ 方法包括**步骤1**、**步骤2**和**步骤3**
```

**规则**：每个需要强调的词分别加粗，不要连续大段加粗。

---

## 加粗的使用场景

### 适合加粗

1. **核心概念**：首次出现的关键术语
2. **重要结论**：punchline、关键发现
3. **数值结果**：显著的数据
4. **方法名称**：核心方法、算法名称

### 不适合加粗

1. **普通描述**：一般性叙述
2. **全文通用词**：反复出现的常用词
3. **长句**：超过10个字的句子
4. **标题**：标题本身已经突出，不需要加粗

---

## 检查方法

### 自动检查

使用`tools/check_blog_quality.py`会自动检测：
- 加粗包含标点符号
- 加粗包含百分号
- 加粗包含引号
- 加粗包含括号

### 手动检查

在编辑器中搜索：
1. `**.*[。，；：！？.]**` - 查找包含中文标点的加粗
2. `**\d+%**` - 查找包含百分号的加粗
3. `**.*（.*）.***` - 查找包含括号的加粗
4. `**".*"**` - 查找包含引号的加粗

---

## 修复工具

`tools/fix_format.sh`会自动修复：
- **24%** → **24**%

但其他情况需要手动修复。

## 常见错误汇总

| 错误写法 | 正确写法 | 错误原因 |
|---------|---------|---------|
| **重要.** | **重要**. | 句号在加粗内 |
| **24%** | **24**% | 百分号在加粗内 |
| **"黑箱"** | "**黑箱**" | 引号在加粗内 |
| **优化（Optimization）** | **优化**（Optimization） | 括号在加粗内 |
| **重要概念！** | **重要概念**！ | 感叹号在加粗内 |
| **这是一个非常非常重要的结论。** | **关键结论**：这是一个非常重要的结论。 | 加粗内容过长 |
